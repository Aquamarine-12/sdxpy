Section 21.2: Displaying a File

  • As mentioned in Chapter 10, screen’s coordinate system is upside down: y=0 is at the top
  • Other than that, ask the screen to display lines and then wait for a quit command
def main(scr, contents):
    scr.erase()
    for i, line in enumerate(contents):
        scr.addstr(i, 0, line)

    while True:
        key = scr.getkey()
        if key in "Qq":
            break
  • Get the file from anywhere
if __name__ == "__main__":
    contents = []
    if len(sys.argv) > 1:
        with open(sys.argv[1], "r") as reader:
            contents = reader.readlines()
    curses.wrapper(main, contents)
  • Works, but only for the right kind of file
  • Try to display a thousand lines, each a thousand characters long
if __name__ == "__main__":
    contents = ['0123456789' * 100] * 1000
    try:
        curses.wrapper(main, contents)
    except Exception as exc:
        print(exc)

addwstr() returned ERR
  • Problem is we’re trying to write outside the window
  • Solution is to trim what we’re displaying
  • Define a Window class to keep track of the display area
    • Why -1?
class Window:
    def __init__(self, nrow, ncol):
        assert 0 <= nrow and 0 <= ncol
        self.nrow = nrow
        self.ncol = ncol
  • Initialize it inside main and then trim what we’re showing
def main(scr, contents):
    win = Window(curses.LINES - 1, curses.COLS - 1)
    scr.erase()
    for i, line in enumerate(contents[:win.nrow]):
        scr.addstr(i, 0, line[:win.ncol])

    while True:
        key = scr.getkey()
        if key in "Qq":
            break

Section 21.3: An Editor Object

  • Functions with lots of parameters are hard to manage
  • But curses.wrapper requires a function
  • Solution: create a class with a __call__ method
    • Must use Delayed Construction design pattern
  • Editor
class Editor:
    def __init__(self):
        self.scr = None
        self.win = None

    def __call__(self, scr, contents):
        self.setup(scr, contents)
        self.interact()
  • Setup
    def setup(self, scr, contents):
        self.scr = scr
        self.win = Window(curses.LINES - 1, curses.COLS - 1)

        self.scr.erase()
        for i, line in enumerate(contents[:self.win.nrow]):
            self.scr.addstr(i, 0, line[:self.win.ncol])
  • Interaction
    def interact(self):
        while True:
            key = self.scr.getkey()
            if key in "Qq":
                break
  • Launch
if __name__ == "__main__":
    editor = Editor()
    contents = ['0123456789' * 100] * 1000
    try:
        curses.wrapper(editor, contents)
    except Exception as exc:
        print(exc)
  • Then add a lookup table of actions (to be fleshed out later)
    def __init__(self):
        self.scr = None
        self.win = None
        self.running = True
        self.actions = {
            "Q": self.quit,
            "q": self.quit
        }
  • Each action must be a method of no arguments (other than self)
    def quit(self):
        self.running = False
