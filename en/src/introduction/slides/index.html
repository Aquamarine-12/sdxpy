--- ## Audience

Maya learned Python as an undergrad in biology. She can create multi-page scripts and notebooks for herself but wants to learn how to build software other people can use. This material will fill in some gaps in her programming knowledge and teach her some common design patterns.

Rupinder is a computer science undergrad. He has learned a lot of theory, and uses Git and unit testing tools in assignments, but doesn't understand how they work. This material will help him understand how software tools work and show him how to design new ones.

Yim teaches a college course on software architecture. They and their students are frustrated that textbooks are so abstract and don't use examples they can relate to. This material will give Yim and their students starting points for a wide variety of course assignments.
